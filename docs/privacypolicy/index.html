プライバシーポリシー

[アンベリー株式会社]（以下「当社」）は、当社が提供する [時差とアラーム]（総じて以下「本アプリ」）およびこれに付随するサービス（以下総称して「本サービス」）における、ユーザーに関する情報の取扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」）を定めます。

第1条（プライバシー情報）

本ポリシーにおいて「プライバシー情報」とは、次の各号に定める情報をいいます。

  • 「個人情報」とは、個人情報の保護に関する法律（個人情報保護法）にいう「個人情報」を指し、生存する個人に関する情報であって、氏名、メールアドレスその他の記述等により特定の個人を識別できるもの、または他の情報と容易に照合することで特定の個人を識別できるものをいいます。
  • 「履歴情報および特性情報」とは、上記の「個人情報」以外のものであり、ユーザーの利用状況・端末情報等に関する情報をいい、たとえば以下を含みます。
    • 本サービスの利用履歴（画面閲覧、機能操作、広告の表示・クリック等）
    • 端末情報（OS、端末モデル、言語設定等）
    • ネットワーク情報（IPアドレス等）
    • クッキー等の識別子、広告ID等の端末識別子
    • クラッシュログ、エラー情報、パフォーマンス情報
    • （ユーザーが許可した場合）通知設定等の権限情報

第2条（プライバシー情報の収集方法）

当社は、本サービスの提供にあたり、ユーザーが外部サービス（Google、Apple等）によるログインを行う場合、当該外部サービスから提供される情報（例：外部サービス上のユーザー識別子、表示名、メールアドレス、プロフィール画像等）を取得することがあります。

当社は、ユーザーが本サービスを利用する際に、利用履歴・端末情報・広告識別子等の履歴情報および特性情報を、自動的に収集することがあります。

当社は、広告配信事業者、解析・クラッシュ解析事業者、購入プラットフォーム事業者等の第三者（以下「提携先」といいます）を利用する場合、ユーザーが本サービスを利用する過程で、提携先が情報を収集することがあります。

第3条（収集・利用する目的）

当社がプライバシー情報を収集・利用する目的は、以下のとおりです。

  • 本サービスの提供、本人確認、ユーザー認証、アカウント管理のため
  • サブスクリプションの購入状況の確認、提供機能の解放、復元対応、課金関連のサポート対応のため
  • 本サービスの運用、保守、障害対応、品質改善、機能改善のため
  • 無料ユーザーへの広告表示、広告の配信・最適化、効果測定のため
  • お問い合わせ対応、重要なお知らせ等の連絡のため
  • 不正利用防止、セキュリティ確保のため
  • 上記利用目的に付随する目的

第4条（第三者提供・外部送信）

当社は、次に掲げる場合を除き、あらかじめユーザーの同意を得ることなく、第三者に個人情報を提供しません。

  • 法令に基づく場合
  • 人の生命、身体または財産の保護のために必要がある場合であって、本人の同意を得ることが困難なとき
  • 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合であって、本人の同意を得ることが困難なとき
  • 国の機関もしくは地方公共団体またはその委託を受けた者が法令の定める事務を遂行することに協力する必要がある場合であって、本人の同意を得ることにより当該事務の遂行に支障を及ぼすおそれがあるとき

前項にかかわらず、当社が本サービス提供のために以下の提携先を利用する場合、ユーザーの端末情報、広告ID、利用履歴等が提携先へ送信されることがあります（外部送信）。

  • ログイン（認証）：Google / Apple 等
  • 広告配信（無料ユーザー）：Google AdMob 等
  • 購入処理（サブスクリプション）：Apple（App Store）等
  • 解析・クラッシュ解析：Firebase / Crashlytics 等（利用している場合）

※利用する提携先は、本サービスの実装・運用状況に応じて変更される場合があります。

当社は、利用目的の達成に必要な範囲で、個人情報の取扱いの全部または一部を委託することがあります（この場合、委託先に対し適切な監督を行います）。

合併その他の事由による事業承継に伴い個人情報が提供される場合があります。

第5条（購入情報について）

本アプリのサブスクリプション購入は、Apple（App Store）等のプラットフォームを通じて行われます。

当社は、購入状況確認や権利提供のために、購入の有効性を示す情報（例：購入状態、取引ID、レシート検証に必要な情報等）を取り扱う場合があります。

クレジットカード番号等の決済情報は、原則としてプラットフォーム事業者が取り扱い、当社が直接取得・保管することはありません。

第6条（広告について）

本アプリの無料ユーザーには、広告を表示する場合があります。

広告配信のため、広告配信事業者が広告ID等の識別子、端末情報、広告表示・クリック等の情報を取得する場合があります。

ユーザーは、OSの設定により広告に関する識別子の制限・リセット等を行える場合があります（端末の仕様によります）。

第7条（個人情報の開示・訂正・削除）

当社は、本人から、当社が保有する個人情報の開示を求められたときは、本人確認のうえ、合理的な範囲で遅滞なく開示します。

当社は、本人から、当社が保有する個人情報の訂正、追加、削除を求められた場合、必要な調査を行い、合理的な範囲で遅滞なく対応します。

ただし、法令により保存義務がある場合、または当社の権利・安全を守るために必要な場合、すべての削除に応じられないことがあります。

第8条（利用停止等）

当社は、本人から、個人情報が利用目的の範囲を超えて取り扱われている、不正の手段により取得された等の理由により利用停止または消去を求められた場合、必要な調査を行い、その結果に基づき合理的な範囲で対応し、その旨を通知します。

第9条（安全管理措置）

当社は、プライバシー情報の漏えい、滅失、毀損、不正アクセス等を防止するため、合理的な安全管理措置を講じます。

第10条（プライバシーポリシーの変更）

当社は、ユーザーに通知することなく、本ポリシーを変更できるものとします。

変更後の本ポリシーは、本ウェブサイトに掲載したときから効力を生じるものとします。

重要な変更がある場合は、本アプリ内または当社Webサイト上で分かりやすい方法により告知します。
